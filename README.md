> Each project below solves one real problem I ran into or got curious about.

**01 · NLP-MultiFeature-Chatbot**
One chatbot, several jobs — medical assistance, research lookup, multilingual chat. `Python` `NLP` `Semantic Search` `GenAI`
The interesting part isn't the chat UI, it's routing a query to the right reasoning mode underneath.
→ [github.com/DileepRajesh-2006/NLP-MultiFeature-Chatbot](https://github.com/DileepRajesh-2006/NLP-MultiFeature-Chatbot)

**02 · roamly-travel**
Turns "I want to go somewhere" into a real itinerary with a budget attached. `JavaScript` `AI Planning`
Treats travel planning as a constraint problem (time, money, interest) rather than a list of links.
→ [github.com/DileepRajesh-2006/roamly-travel](https://github.com/DileepRajesh-2006/roamly-travel)

Predicts personal health risk from input data. `Next.js` `TypeScript` `Supabase`

**04 · musicplayerpro**
